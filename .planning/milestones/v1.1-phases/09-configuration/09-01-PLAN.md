       - `Register(name string, p provider.Provider)` — acquires write lock, adds provider to map
       - `SetDefault(name string)` — acquires write lock, sets defaultProvider
       - `Get(name string) (provider.Provider, bool)` — acquires read lock, returns provider and existence bool
       - `Default() (provider.Provider, error)` — acquires read lock, returns provider for defaultProvider name, or error `fmt.Errorf("default provider %q not found", r.defaultProvider)`

       - `Names() []string` — acquires read lock, returns sorted slice of provider names (use `sort.Strings`)
