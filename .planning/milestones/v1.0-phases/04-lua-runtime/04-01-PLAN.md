    `NewSandboxedState(ctx context.Context) *glua.LState`:
    - Create LState with `glua.Options{SkipOpenLibs: true}`
    - Open safe libs only: LoadLibName/OpenPackage, BaseLibName/OpenBase, TabLibName/OpenTable, StringLibName/OpenString, MathLibName/OpenMath
    - Use `L.CallByParam(glua.P{Fn: L.NewFunction(opener), NRet: 0, Protect: true}, glua.LString(libName))` for each
    - Nil-out unsafe base functions: `L.SetGlobal("dofile", glua.LNil)` and `L.SetGlobal("loadfile", glua.LNil)`
    - Set context via `L.SetContext(ctx)`
    - Preload JSON module: `ljson.Preload(L)` (import `github.com/vadv/gopher-lua-libs/json` as `ljson`)
    - Return L

    **Step 3: Create internal/lua/convert.go**

    `ArgsToLuaTable(L *glua.LState, args api.ToolCallFunctionArguments) *glua.LTable`:
    - Create new table via `L.NewTable()`
    - Iterate `args.All()` (returns `iter.Seq2[string, any]`)
    - Switch on value type:
      - `string` -> `glua.LString(v)`
      - `float64` -> `glua.LNumber(v)`
      - `bool` -> `glua.LBool(v)`
      - `nil` -> `glua.LNil`
      - default -> JSON marshal to string, set as `glua.LString`
    - Use `L.SetField(tbl, key, lvalue)` for each

    **Step 4: Create internal/lua/luatool.go**

    Types:
    ```go
    type LuaParam struct {
        Name        string
        Type        string // "string", "number", "boolean"
        Description string
        Required    bool
    }

    type LuaTool struct {
        name        string
        description string
        params      []LuaParam
        scriptPath  string
        proto       *glua.FunctionProto
    }
    ```

    `CompileFile(path string) (*glua.FunctionProto, error)`:
    - Open file, create bufio.Reader
    - `parse.Parse(reader, path)` to get chunk
    - `glua.Compile(chunk, path)` to get proto
    - Return proto (import `github.com/yuin/gopher-lua/parse`)

    `NewLuaToolFromProto(proto *glua.FunctionProto, scriptPath string) (*LuaTool, error)`:
    - Create a temporary sandboxed LState (with `context.Background()`) to extract metadata
    - Execute the proto: `L.NewFunctionFromProto(proto)`, push, PCall(0, 1, nil)
    - Validate returned value is a table (CheckTable)
    - Validate required fields:
      - `name` must be LTString and non-empty
      - `description` must be LTString and non-empty
      - `execute` must be LTFunction
      - `parameters` if present must be LTTable
    - Extract parameters: iterate the parameters table (integer-keyed), each entry must have `name` (string). Extract `type` (default "string"), `description` (default ""), `required` (default false as LBool).
    - Close the temporary LState
    - Return `&LuaTool{name, description, params, scriptPath, proto}`
    - Return descriptive errors: `fmt.Errorf("%s: missing or non-string 'name' field", scriptPath)`

    `func (lt *LuaTool) Name() string` -- returns lt.name

    `func (lt *LuaTool) Definition() api.Tool`:
    - Create `api.NewToolPropertiesMap()`
    - For each param: `props.Set(p.Name, api.ToolProperty{Type: api.PropertyType{p.Type}, Description: p.Description})`
    - Collect required param names
    - Return `api.Tool{Type: "function", Function: api.ToolFunction{Name: lt.name, Description: lt.description, Parameters: api.ToolFunctionParameters{Type: "object", Required: required, Properties: props}}}`

    `func (lt *LuaTool) Execute(ctx context.Context, args api.ToolCallFunctionArguments) (string, error)`:
    - Create fresh sandboxed state: `L := NewSandboxedState(ctx)`, defer L.Close()
    - Load proto: `fn := L.NewFunctionFromProto(lt.proto)`, push, PCall(0, 1, nil)
    - Get returned table, get "execute" field
    - Convert args: `argsTable := ArgsToLuaTable(L, args)`
    - Call execute function via `L.CallByParam(glua.P{Fn: executeFn, NRet: 1, Protect: true}, argsTable)`
    - Get result: `result := L.Get(-1)`, pop, return `glua.LVAsString(result), nil`
    - If result is LNil, return `"", nil`
    - Wrap all errors with tool name and script path
