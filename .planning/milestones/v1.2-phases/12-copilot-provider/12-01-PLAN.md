Implement two functions:

1. `resolveTokenWith(lookPath func(string) (string, error), command func(string, ...string) ([]byte, error)) (string, error)` — injectable version for testing:
   - Check `os.Getenv("GH_TOKEN")` — return immediately if non-empty
   - Check `os.Getenv("GITHUB_TOKEN")` — return immediately if non-empty
   - Call `lookPath("gh")` — if error, return: `fmt.Errorf("copilot provider requires the GitHub CLI (gh). Install from: https://cli.github.com")`
   - Call `command("gh", "auth", "token", "--hostname", "github.com")` to get token bytes
   - If error: check if it's `*exec.ExitError`. If exit code 4: return `fmt.Errorf("GitHub CLI is not authenticated. Run: gh auth login")`. Otherwise: return `fmt.Errorf("gh auth token failed: %s", strings.TrimSpace(string(exitErr.Stderr)))`
   - `strings.TrimSpace(string(output))` — if empty: return `fmt.Errorf("gh auth token returned empty token")`
   - Return the trimmed token

2. `resolveToken() (string, error)` — production wrapper that calls `resolveTokenWith` with real implementations:
   - `lookPath`: `exec.LookPath`
   - `command`: a closure that creates `exec.Command(name, args...).Output()` and returns `([]byte, error)`

Imports needed: `fmt`, `os`, `os/exec`, `strings`

NOTE: The `command` function signature for `resolveTokenWith` should be `func(name string, args ...string) ([]byte, error)` to match how exec.Command works. The real implementation creates the *exec.Cmd internally.

For the ExitError check in `resolveTokenWith`: since the `command` func returns generic `([]byte, error)`, the ExitError type assertion works because the real exec.Command.Output() returns *exec.ExitError. The mock in tests can return a custom error or nil.

Provider struct:
```go
type Provider struct {
    inner *openaiProvider.Provider
}
```

Constructor `New() (*Provider, error)`:
- Call `resolveToken()` to get the GitHub token
- If error, return `nil, fmt.Errorf("copilot: %w", err)`
- Call `openaiProvider.New(baseURL, token)` to create the inner provider
- If error, return `nil, fmt.Errorf("copilot: creating openai client: %w", err)`
- Return `&Provider{inner: inner}, nil`

- `Name() string` — return `"copilot"` (NOT delegated — this is the copilot identity)
