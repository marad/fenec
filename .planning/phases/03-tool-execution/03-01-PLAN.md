    3. `IsDangerous(command string) bool` -- trims whitespace, checks if command contains any pattern from dangerousPatterns. Returns true if any match.

    Create `internal/tool/shell.go` with:

    1. `ShellResult` struct:
       ```go
       type ShellResult struct {
           Stdout   string `json:"stdout"`
           Stderr   string `json:"stderr"`
           ExitCode int    `json:"exit_code"`
           TimedOut bool   `json:"timed_out,omitempty"`
       }
       ```

    2. `ToJSON() string` method on ShellResult: marshals to JSON. Truncates stdout and stderr to `maxOutput = 4096` characters each, appending `"\n... (truncated)"` if truncated.

    3. `ShellTool` struct with fields:
       - `timeout time.Duration` -- default command timeout
       - `approver ApproverFunc` -- called for dangerous commands (nil means deny all dangerous)

    4. `NewShellTool(timeout time.Duration, approver ApproverFunc) *ShellTool`

    5. `Name() string` -- returns `"shell_exec"`

    6. `Definition() api.Tool` -- returns:
       ```go
       api.Tool{
           Type: "function",
           Function: api.ToolFunction{
               Name:        "shell_exec",
               Description: "Execute a shell command and return stdout, stderr, and exit code. Use this to run programs, inspect the filesystem, or perform system operations.",
               Parameters: api.ToolFunctionParameters{
                   Type:     "object",
                   Required: []string{"command"},
                   Properties: props,  // ToolPropertiesMap with "command" string property
               },
           },
       }
       ```
       Where `props` is built via `api.NewToolPropertiesMap()` then `props.Set("command", api.ToolProperty{Type: api.PropertyType{"string"}, Description: "The shell command to execute"})`.

    7. `Execute(ctx context.Context, args api.ToolCallFunctionArguments) (string, error)`:
       - Extract "command" from args via `args.Get("command")`, type-assert to string. Return error `"missing required argument: command"` if not found or not a string.
       - Check `IsDangerous(command)`. If dangerous and approver is nil, return error `"dangerous command denied: no approver configured"`. If dangerous and approver returns false, return error `"dangerous command denied by user: <command>"`.
       - Call internal `executeShell(ctx, command, s.timeout)`.
       - Return `result.ToJSON(), nil`.

    8. `executeShell(ctx context.Context, command string, timeout time.Duration) (*ShellResult, error)`:
       - Create timeout context: `ctx, cancel := context.WithTimeout(ctx, timeout)`
       - Create command: `cmd := exec.CommandContext(ctx, "/bin/sh", "-c", command)`
       - Set `cmd.WaitDelay = 5 * time.Second`
       - Set `cmd.SysProcAttr = &syscall.SysProcAttr{Setpgid: true}` for process group management
       - Capture stdout and stderr into `bytes.Buffer`
       - Run and handle errors:
         - If `ctx.Err() == context.DeadlineExceeded`: return ShellResult with TimedOut=true, ExitCode=-1
         - If `*exec.ExitError`: extract ExitCode from ProcessState
         - Other errors: return nil, error
       - Return ShellResult with captured stdout, stderr, exit code

    Imports for shell.go: `bytes`, `context`, `encoding/json`, `fmt`, `os/exec`, `syscall`, `time`, `github.com/ollama/ollama/api`
    Imports for safety.go: `strings`
