   - TestRegistryDefaultName: SetDefault("ollama"), assert DefaultName() == "ollama"
   - TestRegistryDefaultNameAfterUpdate: Update with new default, assert DefaultName reflects new value
   - TestRegistryDefaultNameEmpty: New registry, assert DefaultName() == ""

3. Rewrite the `--model` flag handling section in main.go (currently lines ~143-161). The new logic:
   - If `*modelName` contains `/`, split on first `/` via `strings.SplitN(*modelName, "/", 2)`:
     - Left part = provider name. Look up via `providerRegistry.Get(providerName)`.
     - If provider not found, print error with available providers from `providerRegistry.Names()` and exit.
     - Right part = model name. Set `p` to this provider, `defaultModel` to model name.
     - Skip the models list validation here — let the provider return an error at runtime if model doesn't exist. This avoids blocking startup with a ListModels call to a potentially slow provider.
   - If `*modelName` does NOT contain `/`:
     - Keep existing behavior: use default provider `p`, set `defaultModel = *modelName`.
     - Same skip-validation approach: don't iterate models list to check, just set the model name.
   - If `*modelName` is empty:
     - Use `cfg.DefaultModel` if set (non-empty). This falls through from config's `default_model` field.
     - Otherwise keep existing behavior: list models from default provider and use first.

4. Update the `--model` flag help text from "Ollama model to use" to "Model to use (provider/model or just model name)".

5. Remove the current ListModels + validation block for --model. The current code lists all models just to check if the flag value is in the list — this is wasteful. Instead, trust the user's input and let the provider error naturally if the model doesn't exist. Keep the ListModels call that picks the first model when no --model flag is given.

6. Pass `providerRegistry` to NewREPL (handled in Task 2 signature change). For this task, just ensure main.go tracks which provider name is active:
   - Add `activeProviderName` string variable. Set it to `providerRegistry.DefaultName()` initially.
   - When `--model provider/model` overrides, set `activeProviderName` to the parsed provider name.
   - Pass `activeProviderName` to NewREPL (new parameter, added in Task 2).
