       j. Extract 3 lines of context before and after the edit point. Find the index where replacement occurred in the updated content using `strings.Index(original, old_text)` to get the byte position, then count lines up to that point and extract lines [editLine-3, editLine+newTextLines+3].
       k. Return JSON-marshaled `EditResult{Status: "ok", LinesChanged: linesChanged, Context: contextStr}`.
